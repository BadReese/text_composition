# text_composition

flutter 中文排版 分页 上下对齐 两端对齐

## 特点

- [x] 中文英文符号混排两端对齐
- [x] 支持非等高字体
- [x] 支持非等宽字体
- [x] 按容器高度分页
- [x] 上下分散对齐
- [x] 支持标题`title`与样式`titleStyle`
- [x] 支持链接`link`和样式`linkStyle`
- [ ] `link`点击事件
- [ ] 图片
